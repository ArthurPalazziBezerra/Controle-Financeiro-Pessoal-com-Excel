# Controle-Financeiro-Pessoal-com-Excel
No Brasil, o endividamento atinge a maioria das famílias: segundo dados de 2025 da Confederação Nacional do Comércio de Bens, Serviços e Turismo (CNC), cerca de 76,1% das famílias brasileiras estão endividadas — ou seja, mais de três quartos da população vive com contas a vencer. 
Além disso, para uma parcela significativa dessas famílias, o comprometimento com dívidas consome uma fatia elevada da renda mensal: em muitos casos, as famílias destinam 30% ou mais dos seus ganhos apenas para quitar dívidas. 
Esses dados mostram uma realidade alarmante de fragilidade financeira, especialmente entre populações de baixa renda — com risco constante de inadimplência, ausência de reserva de emergência e vulnerabilidade a imprevistos.
É diante desse contexto que surge a proposta do meu projeto: uma planilha de finanças pessoais estruturada em três páginas — uma para registro diário de gastos, outra para controle mensal e uma terceira com dashboard para visualização clara e organizada dos fluxos financeiros. A ideia é oferecer uma ferramenta simples, prática e acessível, que ajude qualquer pessoa — especialmente aquelas em situação mais vulnerável — a retomar o controle das finanças, evitar o endividamento e planejar o futuro com mais segurança.
Com esse projeto, quero contribuir para que o simples ato de anotar e acompanhar gastos se torne um hábito saudável, ajudando a evitar os efeitos nocivos do endividamento crônico e promovendo maior educação financeira desde o cotidiano.
