# Controle-Financeiro-Pessoal-com-Excel
Quase metade dos brasileiros não controla o próprio dinheiro e 47% têm dificuldade em lidar com finanças. Para ajudar nesse cenário, este projeto oferece uma planilha simples e intuitiva com registro diário, visão mensal e um dashboard para melhorar a organização financeira.
